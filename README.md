## Project Overview

This project analyses how lifestyle behaviours and occupational factors influence sleep health outcomes. The aim is to identify high-risk groups and key drivers of poor sleep to inform workplace wellness and preventative health strategies.

## Tools Used

* Power BI
* Power Query
* DAX Functions

## Data Source

The dataset was sourced from Kaggle and includes demographic, occupational, lifestyle, and sleep health indicators.

## Key Focus Areas

* Physical activity and sleep outcomes
* Occupational risk factors
* Stress and sleep health
* BMI and sleep disorder prevalence
